# NodeJS: The complete guide

Code for the Maximilian's ["NodeJS: The complete guide"](https://www.udemy.com/course/nodejs-the-complete-guide/) course.

## Install and run
To run the code. Enter a directory,

```bash
npm install
```

and then

```bash
npm run
```
